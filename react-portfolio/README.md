# React TypeScript Tailwind Portfolio

Minimal scaffold with modular components for Tecnologías, Proyectos y Certificados.

Quick start:

1. cd react-portfolio
2. npm install
3. npm run dev

Personaliza tus datos reales editando `src/data/demoData.ts` (proyectos, tecnologías, certificados y `socialLinks`).

Sección contacto reemplaza el botón de "Descargar CV". Personaliza los enlaces en `src/App.tsx`.
